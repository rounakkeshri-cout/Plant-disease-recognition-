# 🌿 Plant Disease Recognition System

## 🔍 Overview

This project is a Deep Learning-based web application that detects plant diseases from leaf images.
It uses a trained Convolutional Neural Network (CNN) model to classify diseases and help farmers or users take early action.

---

## 🚀 Features

* 📸 Upload leaf image
* 🤖 AI-based disease prediction
* ⚡ Fast and accurate results
* 🌐 Simple web interface (Streamlit)

---

## 🧠 Tech Stack

* Python
* TensorFlow / Keras
* Streamlit
* NumPy, Pillow

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/Plant-disease-recognition.git
cd Plant-disease-recognition
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run main.py
```
